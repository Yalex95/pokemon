Projecto hecho con  [Next.js](https://nextjs.org/)

## Ejecutar el servidor de desarrollo:

```bash
npm run dev
# or
yarn dev
```

Abre [http://localhost:3000](http://localhost:3000) desde tu navegador y listo.
